QR-код для этой страницы

Для изготовления таблички существует много способов. В самом простом и доступном исполнении достаточно распечатать QR код на принтере и поместить его в акриловую квадратную капсулу (обычно используется для изготовления фото магнитов). Простой и недорогой способ - заказать стикеры-наклейки на водостойкой виниловой основе. Другие варианты изготовления таблички - это контрастная лазерная гравировка, УФ-печать, фотопечать. Так же можно нанести QR код на пластиковую или фарфоровую табличку. Обязательным условием для хорошего распознавания гаджетами QR кода является высокая контрастность узора. Современные методы гравировки по камню позволяют нанести QR код непосредственно на стелу. Любой из перечисленных вариантов быстро и качественно для вас помогут изготовить наши партнеры.

Тариф “Галерея памяти”

Вы сможете разместить основную информацию об усопшем:

  • фото основное;
  • даты жизни;
  • обстоятельства смерти;
  • факты из биографии;
  • 3 папки с фотографиями с возможностью сделать подпись под каждой из них;
  • воспоминания друзей;
  • эпитафия.

Папки с фотографиями.
Позволяет пользователю создавать альбомы с фотографиями. Количество альбомов ограничено — 3. Для альбома должна быть предусмотрена возможность добавления названия. Каждый альбом предполагает возможность загрузки до 10 фотографий с указанием подписи к каждой из них. Вес фотографий должен быть ограничен 2Мб.

Каждый структурный элемент страницы памяти заполняется зарегистрированным посетителем сайта самостоятельно в соответствии с правилами сайта. После оплаты и проверки страница публикуется с последующей генерацией QR кода.
